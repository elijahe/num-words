# THIS IS STILL A WORK IN PROGRESS
Convert numbers into words.

## Limitations
* Supports integers with 66 digits or less
* Does not support delimiters or decimal points

# Contributing

## Prerequisites
Node (version 8.9.4 tested)
npm (version 5.6.0 tested)

## Running the tests
`npm test`

## Running eslint
`npm run lint`

## Building
`npm run build`
